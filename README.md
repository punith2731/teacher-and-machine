
# Frontend - Student Chapter System

## Setup

1. Install dependencies:

   ```bash
   cd frontend
   npm install
   ```

2. Run dev server:

   ```bash
   npm run dev
   ```

3. Open the URL printed in the terminal (usually http://127.0.0.1:5173).

Make sure your backend (FastAPI) is running on http://127.0.0.1:8001
